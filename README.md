# 🚦 AI Emergency Traffic Signal System

This project simulates how AI can help emergency vehicles like ambulances get priority in traffic using computer vision.

## 🔍 Features
- Vehicle detection using YOLOv8
- Emergency vehicle simulation
- Automatic traffic signal control logic

## ⚙️ Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## 🚀 How to Run

1. Install dependencies:
pip install ultralytics opencv-python

2. Add a traffic video:
- Place a file named `video.mp4` in the project folder

3. Run the project:
python detect.py

## 💡 Idea
This system demonstrates how AI can dynamically control traffic signals to create a clear path for emergency vehicles.
